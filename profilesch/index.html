2. Cara Instalasi

  1. Masuk ke Dashboard Blogger.
  2. Pilih menu Tema → klik ikon panah ▼ → Pulihkan.
  3. Upload file profiesch.xml dari paket ZIP tema.
  4. Simpan
  5. Masuk ke Tata letak / Layout → scroll kebawah sampai kolom Lisensi→ masukkan lisensi→ simpan
  6. Selesai - Tema siap digunakan / disesuaikan

3. Pengaturan Tema

Berikut ini langkah - langkah untuk mengatur tema Profile Sch. Pastikan mengikuti panduan agar tema berjalan optimal.

1. Pengaturan Label Wajib

Tema ini menggunakan sistem label (kategori) untuk menampilkan konten secara terstruktur pada bagian-bagian tertentu. Pastikan Anda membuat dan menggunakan label berikut agar fitur tema berjalan dengan baik:

  • Artikel — Label utama untuk semua postingan reguler atau berita blog. Label ini bisa digabung dengan label lainnya untuk postingan blog.
    "Label Artikel Jangan digabung dengan Data Guru dan Gallery"
  • Data Guru — Label khusus untuk menampilkan daftar atau profil guru di bagian tertentu.
  • Gallery — Label khusus untuk menampilkan gambar atau galeri di halaman galeri.

Cara Menambahkan Label Saat Menulis Postingan

  1. Buka menu Postingan → Postingan Baru
  2. Di sisi kanan (sidebar), cari bagian Label
  3. Ketik nama label sesuai: Artikel, Data Guru, atau Gallery
  4. Gunakan hanya satu label wajib untuk tiap kategori, agar tidak tercampur.
  5. Publikasikan postingan seperti biasa.

Catatan: Label bersifat case-sensitive, jadi penulisan harus sesuai (misal: "Gallery", bukan "gallery" atau "GALERI").
Anda juga dapat membuat / menambahkan label lainnya.

2. Navigasi Menu Utama

2.1 Kode Navigasi Menu Utama

Berikut ini adalah struktur menu navigasi utama yang digunakan dalam tema ini. Menu ini mendukung dropdown, mega menu, dan responsif mobile dengan toggle. 

<li><a href='/'>Beranda</a></li>

<li>
  <a class='desktop-item' href='#'>Profile</a>
  <input id='showDrop' type='checkbox' class="show-toggler" />
  <label class='mobile-item' for='showDrop'>Profile</label>
  <ul class='drop-menu'>
    <li><a href='/p/profil-sekolah.html'>Tentang Profile Sch</a></li>
    <li><a href='/p/sambutan-kepala-sekolah.html'>Sambutan Kepala Sekolah</a></li>
    <li><a href='/search/label/Data%20Guru'>Data Guru dan Staf</a></li>
    <li><a href='#'>Visi dan Misi</a></li>
  </ul>
</li>

<li><a href='/search/label/Gallery'>Gallery</a></li>
<li><a href='#'>Contact</a></li>
<li><a href='/search/label/Artikel'>Blog</a></li>

Catatan: Gunakan struktur kode tersebut untuk membuat menu, jangan merubah nama class dan id agar tampil sesuai pengaturan tema.

  • Kode ini dirancang khusus untuk dipasang di “Tata Letak” → “Menu” dalam Blogger.

4. FAQ (Pertanyaan Umum)

  • Gambar thumbnail tidak muncul?
    Pastikan ada <img> di awal isi postingan.
  • Komentar tidak muncul?
    Cek setelan komentar dan pastikan blog tidak privat.
  • Bagaimana ganti warna?
    Edit CSS di bagian :root atau gunakan Desainer Tema.
